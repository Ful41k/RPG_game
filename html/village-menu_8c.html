Detailed Description

Implementation of the central hub and Blacksmith interactions.

  • Acts as the primary state manager for the player's loop. It controls access to missions based on time of day and handles permanent player upgrades through the Blacksmith.

◆ enterVillage()

void enterVillage ( Player * hero)

The main loop for village interaction.

Enters the main village square loop.

  • This function handles the "Day/Night" cycle constraints:
  • Missions: Only available at Night.
  • Blacksmith: Only available during the Day.
  • Tavern: Always available (but menu changes internally).
  • Waiting: Allows manual transition from Day to Night.
  • Parameters
    heroPointer to the player structure.
