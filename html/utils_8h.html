void print_typewriter ( const char * text,
int speed )

Character-by-character text effect.

Character-by-character text effect.

  • Logic:
  1. Iterates through the string.
  2. Checks kbhit_check for user input; if found, switches to "skip mode."
  3. Detects ANSI escape sequences (starting with '\x1b') to print them instantly, preventing broken color codes during the animation.
  • Parameters
    textThe string to print (supports ANSI colors).
    speedDelay in milliseconds between characters.
