Displays available saves and handles Loading or Deletion.

  • Features:
  1. Scan: Loops and reads all "save_X.dat" files to display stats.
  2. Load: Reads the binary data directly into the hero struct.
  3. Delete: Removes a file and reorganizes the remaining files (e.g., if you delete slot 2, slot 3 is renamed to 2) to maintain a continuous list.
  • Parameters
    heroPointer to the player structure to be populated.
    Returns
    1 if a game was loaded, 0 if deleted or cancelled.
