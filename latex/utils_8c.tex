\doxysubsection{Detailed Description}
Cross-\/platform terminal utilities for text rendering and input. 


\begin{DoxyItemize}
\item This module provides abstraction for Windows and Unix-\/based systems (Linux/mac\+OS). It handles the "{}\+Typewriter"{} effect, screen clearing, and non-\/blocking input detection used to skip text animations. 
\end{DoxyItemize}
